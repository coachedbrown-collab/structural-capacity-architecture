# 3-KPI Tracking Rule

## Overview
We're not going to track ten, we're going to track three. We're going to move one at a time because the one we move two or three others.

## Core Principle
Focus on the constraint. Moving the primary bottleneck KPI pulls the others forward. Tracking too many metrics dilutes focus and creates noise.

## Implementation
1. Identify the single structural constraint.
2. Select three KPIs that directly measure progress on that constraint.
3. Track sequentially. Move one, watch the others move.
4. Review weekly as part of KPI Governance.
